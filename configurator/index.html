Deployment checklist

      

      
    
        
  1. Drop the bundle on your host
    Unzip it into a folder. docker-compose.yml and .env must sit together.
    2. 
        
  2. Bring it up
    Run docker compose up -d, then docker compose logs -f to watch the Fika install and the server boot.
    3. 
        
  3. Open the port
    The server listens on your chosen port (default 6969). Open it on your firewall/router if friends connect over the internet.
    4. 
        
  4. Connect
    Point the SPT launcher at https://your-server-ip:6969 and raid.
    5. 
        
  5. Back up before updates
    Your data directory holds profiles and mods — copy it before changing SPT versions.

Briefing

        
Questions

      

      

        
Do I need to own Escape from Tarkov?
No game files are bundled or required to run the server. SPT is a single-player/co-op server stack — see the SPT and Fika projects for client setup.

        
Where do my profiles and mods live?
In the data directory you set (default ./server-data), bind-mounted into the container. They persist across restarts and updates.

        
How do I update SPT or Fika?
For SPT, pull a new image tag and docker compose up -d. For Fika, set AUTO_UPDATE_FIKA and bump the version — it reinstalls in place, keeping your config.

        
Can friends join over the internet?
Yes — keep "Listen on all networks" on and forward your game port. Each player still installs the SPT launcher and Fika client.

        
x86 or ARM?
The image is multi-arch, so the server runs on either. The headless client is x86-only, so the configurator hides it when you pick ARM.

        
What does the headless client do?
It's an optional Fika client that runs unmanned to host raids, so players can join even when nobody's hosting. It's x86-only and off by default — turn it on in the HEADLESS tab.

        
How do I add other mods?
Paste archive URLs (.zip or .7z) into the MODS tab, one per line. Each is downloaded once on boot and installed; the archive must contain a user/ and/or BepInEx/ folder at its root.

        
Does anything I enter get uploaded?
No. This is a static page with no backend — your options never leave the browser, and the bundle is built locally. You can even run it offline.

        
How are my profiles backed up?
SPT 4.0 backs up profiles itself. For extra safety, copy your data directory before changing SPT versions — it holds your profiles, mods and configs.
