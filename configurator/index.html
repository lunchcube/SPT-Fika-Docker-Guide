Deployment checklist

      
    
        
  • Run docker compose up -d, then docker compose logs -f to watch the Fika install and server boot.
    • 
        
  • The server listens on your chosen port (default 6969). Open it on your firewall / router if friends connect over the internet.
    • 
        
  • Server files live in your data directory — back it up before changing SPT versions.
    • 
        
  • To reconfigure: change options here, re-download, replace the files, and docker compose up -d again.
