# configurator/

The Next.js web configurator (Phase 3). Scaffolded from `Dildz/Portfolio-Page`.

Form-driven UI that emits a `docker-compose.yml` + `.env` bundle from form state — no backend, static export, deployed behind the shared Caddy proxy.

See **[../DESIGN.md](../DESIGN.md)** §6b (repo role) and §9 (UX). Not yet implemented.
